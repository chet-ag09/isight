Welcome to ISight.

AI Object Detector for Blind and Visually Impaired

About

The app uses Computer Vision to detect objects and identify them. The AI was built using Google's Teachable Machine and we used HTML, CSS and JavaScript to build the Front-End. Our Mission is to enable blind and visually imapired to see through their devices and know more about the world around them.

Made for a hackathon.
